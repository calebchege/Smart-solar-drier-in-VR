# Smart Hybrid Solar Dryer
# Virtual Mechatronic System with PLC Integration and VR Simulation


The Smart Hybrid Solar Dryer is a digitally simulated mechatronic system designed to model and analyze an intelligent agricultural drying solution. The project integrates:

Renewable energy systems (solar PV + thermal)

Industrial automation using PLC (Siemens S7-1500)

Closed-loop control (PI/PID)

Real-time communication between control logic and simulation

Immersive visualization using Unity (VR-enabled)

The system acts as a high-fidelity digital prototype for efficient, controlled drying of agricultural produce.

#  System Architecture
The system is composed of three main subsystems:
# Thermal Subsystem
Cascade solar water heaters

Water-to-air heat exchanger

Electric heating backup

Controlled airflow into drying chamber

# Power Subsystem
Photovoltaic panels

Hybrid inverter (solar, battery, grid)

Battery storage

Single-phase AC backup

# Control Subsystem
Siemens S7-1500 PLC

Sensors:

Temperature

Humidity

Flow

Water level

Solar irradiance (pyranometer)

Actuators:

Pump (DOA25/60)

Solenoid valves

Butterfly valve (airflow control)

Fans

Heating element

## Virtual Environment
Built in Unity

3D models created in Blender

Features:

Real-time sensor visualization

Animated actuators

Dynamic weather simulation (Skybox)

Interactive VR experience
